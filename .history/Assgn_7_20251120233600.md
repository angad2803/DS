## Question 1: Create MARKS Matrix and Perform Operations

### Create matrix of 20 students in 3 subjects

```r
MARKS <- matrix(sample(40:60, 60, replace = TRUE),
                nrow = 20,
                ncol = 3)
colnames(MARKS) <- c("SUB1", "SUB2", "SUB3")
```

### a. Calculate total marks of each student

```r
grand_total <- apply(MARKS, 1, sum)
```

### b. Append total to dataset

```r
MARKS_df <- as.data.frame(MARKS)
MARKS_df$Total <- grand_total
```

### c. Calculate standard error for each subject

```r
st.err <- function(x) {
  sd(x) / sqrt(length(x))
}

stderr_SUBS <- apply(MARKS_df[, 1:3], 2, st.err)
print(stderr_SUBS)
```

### d. Add 0.25 bonus marks to all subjects

```r
MARKS_df[, 1:3] <- MARKS_df[, 1:3] + 0.25
```
